# Lynn Nguyen About Page

This repository hosts a lightweight GitHub Pages site for Lynn Nguyen's recruiter-facing about page.

## Edit content quickly
- Main content and sections: `about.html`
- Visual design/theme: `styles.css`
- Scroll reveal behavior: `script.js`

```bash
python3 -m http.server 4173
```
